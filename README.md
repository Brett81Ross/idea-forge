# 🥽 Idea Forge — Vision Pro Edition

A spatial, Vision Pro–inspired invention lab where users can submit, explore, and share ideas in a futuristic glass-style interface.

This project turns GitHub itself into a lightweight backend using **GitHub Issues as a database**.

---

## ✨ Features

- 🧠 Submit invention ideas instantly
- 📡 Live idea feed powered by GitHub Issues
- 💬 Built-in comments (GitHub-native)
- 🔗 Public idea pages via GitHub
- 🥽 Vision Pro–style glass UI design
- 🌌 Ambient glowing spatial background
- 📱 Fully responsive (mobile + desktop)
- 🚫 No backend required

---

## 🧱 Tech Stack

- HTML (single file app)
- CSS (glassmorphism / Vision Pro UI style)
- JavaScript (vanilla)
- GitHub REST API (data layer)
- GitHub Pages (deployment)

---

## 🚀 How It Works

Instead of using a database:

- Each idea = a **GitHub Issue**
- Comments = issue comments
- Votes (future upgrade) = GitHub reactions
- Storage = GitHub repo itself

This makes the project:
- Free
- Scalable for small usage
- Easy to deploy
